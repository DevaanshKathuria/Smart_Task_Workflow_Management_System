# Smart Task & Workflow Management System (STWMS)

## Project Overview
Smart Task & Workflow Management System (STWMS) is a full-stack web application designed to help teams efficiently manage projects, tasks, and workflows. The system provides role-based access control and supports task lifecycle management from creation to completion.

The backend is built using Node.js, Express, and TypeScript, following clean architecture and software engineering best practices. The frontend is built using React and communicates with the backend through RESTful APIs.

## Scope
- User authentication and authorization using JWT
- Role-based access control (Admin, Manager, Employee)
- Project creation and management
- Task assignment and tracking
- Workflow status management
- Notifications for task updates

## Key Features
- Secure login and registration
- Role-based dashboards
- Create, update, and delete projects
- Assign tasks to employees
- Track task status (To Do, In Progress, Review, Completed)
- Notification system for task-related updates
- REST API-based backend

## Target Users
- Software development teams
- Project managers
- Organizations managing internal workflows

## Non-Functional Requirements
- Scalable backend architecture
- Secure and maintainable codebase
- Follows OOP, SOLID, and design pattern principles
- Proper error handling and logging
